What Is an Executable?

An executable is a file the operating system can load into memory and run as instructions. This guide focuses on one idea: how source code becomes something your OS can actually execute.

Quiz score: 0/3

Core Concept

A source file like main.c or app.py is text. The CPU cannot run text directly. It runs machine code bytes. An executable packages that machine code with metadata so the OS knows how to run it.

Compile step: converts source code into machine code.
Link step: connects your code to required libraries/functions.
Load step: OS loader maps executable segments into memory and jumps to the entry point.

One File, Different Formats

Executable formats vary by OS: ELF (Linux), PE (Windows), Mach-O (macOS). Same idea, different headers and loader rules.

Header: tells the OS what architecture this binary targets.
Sections/segments: code, constants, and writable data live in separate regions.
Entry point: first address where execution begins.
